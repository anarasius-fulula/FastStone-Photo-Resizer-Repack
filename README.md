# FastStone Photo Resizer

## Overview

FastStone Photo Resizer is a Windows-based application designed for batch image conversion, resizing, renaming, and editing. It facilitates efficient management of large numbers of photos, enabling users to perform various operations quickly and systematically.

## Key Features

**Batch conversion and resizing of image files**  
**Support for multiple image formats including JPEG, PNG, BMP, TIFF, and GIF**  
**Image renaming with customizable patterns**  
**Basic image editing functions such as cropping, rotating, and color adjustments**  
**Preview feature to review images before processing**  
**Command line support for automation and scripting**  
**User-friendly interface optimized for straightforward workflows**
